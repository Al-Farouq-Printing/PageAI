# PageAI by Hasnex — دليل التشغيل (نسخة HTML بالكامل، بدون سيرفر)

كل حاجة دلوقتي شغالة من داخل المتصفح مباشرة — تسجيل الدخول، توليد المواقع بالذكاء الاصطناعي، إشعارات تليجرام، النشر — كله في ملفات HTML/JS بسيطة، ترفعيها على GitHub Pages وخلاص.

## الملفات
```
index.html      الرئيسية
pricing.html     الأسعار
about.html       من نحن
contact.html     تواصل معنا
login.html       تسجيل الدخول
dashboard.html    لوحة توليد المواقع (أهم صفحة)
view.html        صفحة عرض المواقع المنشورة (رابط عام)
firebase-init.js  إعدادات Firebase + المفاتيح + دوال تسجيل الدخول
firestore.rules   قواعد حماية قاعدة البيانات
```

---

## ⚠️ تنبيه مهم قبل ما تبدأي (اقرأيه مرة واحدة وكفاية)

بما إن مفيش سيرفر خالص، **مفتاح Gemini وتوكن تليجرام ظاهرين لأي حد يفتح كود الصفحة** (زر الفأرة اليمين → View Page Source). ده مش خطأ في الكود، ده نتيجة طبيعية لعدم وجود سيرفر. اللي ممكن يحصل لو حد شرير شافهم:
- ياخد مفتاح Gemini ويستخدمه هو كمان → فاتورتك بتاعة Gemini ممكن تزيد
- يبعت رسائل من بوت التليجرام بتاعك (مش هيقدر يوصل لحسابك الشخصي أو حاجة تانية، بس هيزعجك برسائل)

لو ده مقلقك مستقبلاً لما الموقع يكبر، الحل الوحيد وقتها هو سيرفر بسيط (زي اللي كان متعمول على Netlify قبل كده). حاليًا للمرحلة دي، الموضوع مقبول.

## الخطوة 1: مشروع Firebase

1. `console.firebase.google.com` → افتحي مشروعك (أو اعملي واحد جديد سميه PageAI)
2. **Build → Authentication → Get started** → فعّلي **Google** و **Email/Password**
3. **Build → Firestore Database → Create database** (Production mode)
4. من ⚙️ **Project settings → General → Your apps → </> Web** → سجّلي تطبيق ويب لو لسه معملتيش، وانسخي بيانات الـ config
5. الصقيها في `firebase-init.js` **بالسطور من 6 إلى 13 بالظبط** (بدّلي كل القيم اللي بين علامتي تنصيص بالقيم الحقيقية)

---

## الخطوة 2: نشر قواعد Firestore

Firebase Console → **Firestore Database → Rules** → الصقي محتوى ملف `firestore.rules` بالكامل → **Publish**

---

## الخطوة 3: رفع المشروع على GitHub Pages

1. اعملي Repository جديد (اختاري **Private** — أهم خطوة بما إن المفاتيح جوه الكود)
2. ارفعي كل الملفات
3. **Settings → Pages → Source → main branch** → هيديكي رابط `username.github.io/repo-name`

---

## الخطوة 4: أضيفي دومين GitHub في Firebase

Firebase Console → **Authentication → Settings → Authorized domains → Add domain** → حطي `username.github.io` بالظبط (من غير https://)

1. العميل يضغط "اشتراك" في لوحة التحكم → يوصلك إشعار تليجرام فيه بريده و UID بتاعه
2. يحوّل فودافون كاش على **01070387240**
3. بعد ما تتأكدي من التحويل: افتحي Firebase Console → **Firestore Database → Data → users → (UID بتاع العميل)**
4. غيّري الحقل `plan` لـ `"pro"` أو `"business"`، وحقل `subscriptionExpiry` لتاريخ بعد 30 يوم من دلوقتي (كـ Timestamp)
